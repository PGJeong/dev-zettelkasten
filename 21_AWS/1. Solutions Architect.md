# 도구 및 기술

- **핵심 설계 도구**
	- **AWS 콘솔** - 웹 기반 관리 인터페이스
	- **AWS CLI** - AWS 서비스 관리용 명령줄 인터페이스
	- **AWS CloudFormation** - 템플릿 기반 리소스 프로비저닝을 사용하는 코드형 인프라(IaC) 서비스
	- **AWS CDK(Cloud Developement Kit)** - 개발자가 잘 알고 있는 프로그래밍 언어를 사용하는 코드형 인프라
	- **AWS 다이어그램 및 아키텍처 센터** - 참조 아키텍처 및 모범 사례

- **인프라 계획 및 관리**
	- **AWS Well-Architected Tool** - 아키텍처를 AWS 모범 사례와 비교 검토
	- **AWS Trusted Advisor** - AWS 환경 최적화를 위한 실시간 지침
	- **AWS Cost Explorer** - 비용 예측 및 최적화 도구
	- **AWS Organizations** - 계정 관리 및 통합 결제
	- **AWS Control Tower** - 다중 계정 환경 설정 및 관리

- **모니터링 및 보안**
	- **AWS CloudWatch** - 모니터링 및 관찰성 제공 서비스
	- **AWS CloudTrail** - 거버넌스, 규정 준수 및 감사 기능 제공
	- **AWS Config** - 리소스 인벤토리 및 구성 기록 추적 기능 제공
	- **AWS Security Hub** - 보안 태세 관리

- **배포 및 DevOps**
	- **AWS CodePipeline/AWS CodeDeploy/AWS CodeBuild** - 지속적인 통합/배포
	- **AWS Elastic Beanstalk** - 애플리케이션 배포용 서비스형 플랫폼(PaaS)

- **서드 파티 도구**
	- **Terraform** - 다중 클라우드 코드형 인프라(IaC) 플랫폼
	- **Draw.io 또는 Lucidchart** - 아키텍처 다이어그램 작성 도구
	- **Git 리포지토리 -** 인프라 코드 버전 제어
